Tips for Star Wars: Battlefront II (2005) on PC

LAN gaming through VPN - Freebox OpenVPN server example

  • Set up the OpenVPN server (bridged mode) on the Freebox
  • Download the .ovpn files corresponding to the user created
  • Edit and append to the file: 'ifconfig 192.168.0.A 255.255.255.0', with A being a valid local IP address
  • Add: 'script-security 2'
  • Add: 'up route.sh', with 'route.sh' a script given below
  • sudo openvpn [file].ovpn

route.sh:

#!/bin/sh
route add -host 255.255.255.255/32 tap0

Command line arguments

A list of command line arguments for 'BattlefrontII.exe':

  • /nointro - skips the intro
  • /win - windowed
  • /resolution 320 240 - maximum recommended: 1280 720
  • /[no]framelock - disable V-Sync
